# AI Recipe Generator

An interactive web application that generates unique recipes using Artificial Intelligence based on ingredients or prompts provided by the user.

##  Live Demo
👉 https://ai-recipe-generator-48v0.onrender.com/

# Overview
The AI Recipe Generator allows users to input ingredients or a recipe idea and instantly receive a creative, AI-generated recipe. This project demonstrates the integration of AI into real-world applications to enhance user experience and productivity.

# Features
- Generate recipes using AI
- User-friendly interface
- Responsive design (mobile & desktop)
- Real-time results
- Clean and modern UI

# Tech Stack
- HTML
- CSS
- JavaScript
- AI API (SheCodes API)

# How It Works
1. The user enters ingredients or a prompt  
2. The input is sent to an AI API  
3. The AI processes the request and generates a recipe  
4. The recipe is displayed instantly on the screen  

# Purpose of the Project
This project was built to explore how Artificial Intelligence can be integrated into web applications to create dynamic and personalized user experiences.

# What I Learned
- How to integrate AI APIs into web applications  
- Handling user input and API responses  
- Improving UI/UX design  
- Writing clean and maintainable JavaScript code
